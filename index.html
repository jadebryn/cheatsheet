Jade's Cheatsheet

Semantic HTML

Section Descriptions

Tag Name Description Placement
<header> Header Often contains a logo, the navigation links, important buttons or a search bar Always at the top of the body section
<main> Main Contains all the main content of the page Between the header and footer
<footer> Footer May contain navigation links, author, copyright information, contact information or "back to top" links Bottom of the body section
<nav> Navigation Houses the sites navigational links Usually placed in the header or footer
<figure> Figure Used to hold media such as an image or diagram. The tag <figcaption> is used within it Wherever you want your media
<section> Section Defines elements such as a chapter or heading, used to define the theme for the website Surround your defined section with the tag
<article> Article Holds content that makes sense on it's own (a blog, article, or comment) Sometimes used within a section to write independent content for that theme, but not always
<aside> Aside Marks additional information not needed to understand the main content Often appears in a sidebar or away from the main content

Embedding Media Semantically

Tag Name Description Placement
<video> Video Used to add a video to your site, using the src attribute Wherever you'd like your video
<audio> Audio Used to add some audio to your site, using the src attribute Wherever you'd like your sound
<embed> Embed Used to add any media to your site, using the src attribute Wherever you'd like your media
